# ThreadLens

Paste any Reddit thread URL and get an instant AI-powered breakdown of what people actually think — consensus, praise, complaints, and the contrarian take nobody talks about.

## Tech Stack
- Frontend: React + Vite + TailwindCSS
- Backend: Node.js + Express.js
- Database: MongoDB Atlas
- AI: Google Gemini 1.5 Flash API
- Auth: JWT + bcrypt
